# Apple News
A simple Apple news feed in which you can view short information about the news and, if desired, you can view the news web page in the application, and you can also search for news.
## Features:
- Pagination
- Without using external libraries
- Short information about the news
- Opening a news web page in the app
- Loading images from the cache
- News search
